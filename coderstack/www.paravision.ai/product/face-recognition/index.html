Herramientas Avanzadas

Temario Completo

01

Fundamentos Back-End

Conceptos básicos de programación en el lado del servidor, instalación y configuración de un servidor web e introducción a Node.js

02

API-REST

Fundamentos de REST, reación de endpoints y rutas, implementación de métodos HTTP (GET, POST, PUT, DELETE), trabajo con parámetros de consulta y cuerpo de solicitud y manejo de errores y respuestas.

03

Bases de Datos

Introducción a MongoDB, creación de colecciones y documentos, operaciones CRUD con MongoDB, integración de MongoDB con la API REST e implementación de autenticación y autorización.

04

HTML y CSS

Introducción a HTML, creación de estructuras de página web, introducción a CSS y diseño y estilo de páginas web.

05

Javascript

Introducción a JavaScript, conceptos básicos de programación en JavaScript, manipulación del DOM, eventos, funciones y trabajo con arrays y objetos.

06

ReactJS

Introducción a ReactJS, creación de componentes y vistas, integración con APIs REST, manejo de estado y props y creación de aplicaciones escalables y performantes.

07

Herramientas Avanzadas

Optimización y rendimiento de aplicaciones, uso de herramientas como Babel y Webpack, implementación de pruebas y debugging, uso de librerías y paquetes externos y despliegue de aplicaciones en producción.

Te preparamos para que seas un recurso valioso.
